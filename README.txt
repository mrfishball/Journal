LOCAL RUN:
	-Prerequisites:
		Google App Engine SDK.
		Python 2.7 IDE (Do not download anything higher).
		A Text Editor (I used Sublime. For making changes to the application).

	-Preparation:
		Install Google App Engine SDK and Python 2.7 IDE.
		Open Google App Engine Launcher and go to file then choose "Add Existing Application".
		Select the application that you want to run from the window and hit "Run".
		Application url is localhost:[port]. (Replace [port] with the number you see in the launch window under "port").
		Open a browser and type in the application url. 
		You should see the app running.
